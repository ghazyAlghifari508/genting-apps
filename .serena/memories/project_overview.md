# Project Overview: GENTING
**GENTING (Generasi Anti Stunting)** is a maternal/child health management platform.
Purpose: Prevent stunting through education, monitoring, and AI-driven nutrition analysis.

# Tech Stack
- **Framework**: Next.js 16.1.6 (App Router)
- **Language**: TypeScript
- **Runtime**: Node.js
- **UI**: React 19, Tailwind CSS 4, Framer Motion, Lucide React, Shadcn UI
- **Backend**: Supabase (Auth, DB, Realtime)
- **AI**: Vercel AI SDK (@ai-sdk/openai), OpenRouter
- **Payment**: Midtrans

# Key Dependencies
- `ai`: ^6.0.91
- `@ai-sdk/openai`: ^3.0.30
- `@supabase/supabase-js`: ^2.95.3
