# Coding Conventions
- **Naming**: camelCase for variables/functions, PascalCase for components.
- **Routing**: Next.js App Router (folders in `app/`).
- **Styles**: Tailwind CSS 4 (utility-first).
- **AI Integration**: Custom provider in `lib/openrouter.ts`, API routes in `app/api/ai/`.
- **Database**: Supabase client for data fetching/persistence.
